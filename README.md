🧭 CompassCalibrator — компас с калибровкой
Интерактивный компас с поддержкой калибровки, отображением азимута, сторон света и графической визуализацией.
Реализован на 7 языках программирования для демонстрации подходов к работе с сенсорами и пользовательским интерфейсом.

🧠 Концепция
CompassCalibrator — это не просто компас. Это инструмент для точного определения направления с возможностью калибровки:

✅ Отображение текущего азимута (градусы от 0 до 360).

✅ Визуализация сторон света (N, S, E, W) и промежуточных направлений.

✅ Графический компас с вращающейся стрелкой (в GUI-версиях).

✅ Калибровка — усреднение показаний, учёт магнитных помех (имитация).

✅ Ручной ввод азимута (для тестирования).

✅ Сохранение/загрузка калибровочных коэффициентов.

✅ Симуляция движения компаса (автоматическое вращение) для демонстрации.

✅ Цветовая индикация — красная стрелка на север, зелёная на юг и т.д.

✅ Горячие клавиши (в GUI) для быстрого доступа к функциям.

🚀 Как запустить
Для работы с реальным компасом потребуется аппаратное обеспечение (магнитометр). В тестовом режиме используется симуляция с помощью генератора случайных чисел или ввода пользователя.

bash
# Python (Tkinter)
pip install tk  # обычно уже установлена
python compass_python.py

# C++ (Qt)
qmake && make
./compass_cpp

# Java (Swing)
javac compass_java.java && java compass_java

# C# (WPF)
dotnet run

# Go (консоль)
go run compass_go.go

# Rust (консоль)
cargo build --release && ./target/release/compass_rs

# JavaScript (Node.js)
node compass_js.js

📦 Содержимое репозитория
Файл	Язык	Особенности
